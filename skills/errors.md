# مهارة: معالجة الأخطاء والمزالق في لغة ص

## معالجة الأخطاء البنيوية
```sad
حاول
    ارمي "رسالة خطأ"          # أو: ارمي جديد خطأ_مخصص("تفاصيل")
امسك خ                        # بدون أقواس — الأقواس لم تعد مدعومة
    اطبع_سطر(خ)
أخيراً
    # تنظيف اختياري
نهاية
```
التقاط حسب النوع:
```sad
حاول
    ارمي جديد خطأ_شبكة("انقطاع")
امسك خطأ_شبكة خ          # نوع محدّد أولًا
    # معالجة شبكة
امسك خ                    # عام
نهاية
```

## العقود (Design by Contract)
```sad
دالة قسمة(أ، ب)
    يتطلب ب != 0           # شرط مسبق
    يضمن النتيجة >= 0      # شرط لاحق — النتيجة = القيمة المُرجعة
    ارجع مطلق(أ / ب)      # مطلق ضروريّ، وإلا انتهك الضمان مع مدخلات سالبة
نهاية
```
رتّب `يتطلب` قبل `يضمن`. فشل الشرط يرمي RuntimeError مع نص الشرط.
> انتبه: لا تكتب ضمانًا (`يضمن`) لا يحقّقه جسم الدالّة لكلّ المدخلات الصحيحة — هنا بدون
> `مطلق` يفشل `النتيجة >= 0` عند `أ` سالب رغم استيفاء الشرط المسبق.

## أكثر 20 مزلقًا شيوعًا (وإصلاحها)

1. **نسيان `نهاية`** — كل `دالة/صنف/إذا/بينما/لكل/حاول/طابق/حالة/بنية` تحتاج `نهاية`.
2. **أقواس معقوصة `{}`** — غير مدعومة؛ استعمل `نهاية`.
3. **`return` بدل `ارجع`** — استعمل الكلمات العربية.
4. **نسيان `جديد`** — `جديد حيوان("قط")` لا `حيوان("قط")`.
5. **استدعاء باني الأب** — الكلمة الصحيحة `الأساس` (أو `أساس`)، لا `أصل`.
6. **`وإلا` بعد `نهاية`** — لا تُغلق كتلة `إذا` قبل `وإلا`.
7. **`break`/`continue`** — استعمل `اخرج`/`توقف` و`تابع`/`استمر`.
8. **تعديل ثابت** — `ثابت` لا يُعدّل (حتى `+=`).
9. **نسيان `صدّر`** — الدوال غير المصدّرة غير مرئية من خارج الوحدة.
10. **`لكل` بلا `في`** — `لكل ع في [...]`.
11. **عضو ساكن عبر `هذا`** — استعمل `اسم_الصنف.الحقل`.
12. **خلط كود علوي مع `دالة رئيسية()`** — عند وجود `رئيسية`، كل الجمل التنفيذية داخل دوال.
13. **توقّع تعديل مصفوفة مُمرَّرة لدالة حرة** — تُمرَّر بالقيمة؛ استعمل `&` أو صنفًا.
14. **نسيان `!` للماكرو** — `سجّل!(...)`؛ والتعريف قبل الاستدعاء.
15. **تجاوز دالة موجودة عبر `امتداد`** — ممنوع؛ سمِّها باسم جديد.
16. **المعامل المتغير `...` ليس الأخير** — `(أول، ...باقي)`.
17. **اللامدا ككتلة** — السهم فقط: `(س) => س * 2`.
18. **`امسك (خ)` بأقواس** — استعمل `امسك خ` بدون أقواس.
19. **بنية بباني/دالة** — البنية حقول فقط.
20. **`هدم` بلا أقواس** — `هدم()`.

### مزالق المكتبة (أسماء دقيقة)
- المصفوفة: `احذف_اخير()` (لا `احذف_الاخير`) · `اقلب()`/`قلب()` (لا `اعكس`) · `صل("، ")` (لا `دمج`).
- النص: `يحتوي(...)` (لا `يحتوي_على`) · `يبدأ_بـ`/`ينتهي_بـ` (بتاء مربوطة + ألف صغيرة).
- لا تقطيع للنصوص بالنطاق `نص[0..5]` → "String index must be integer"؛ استعمل حلقة.
- المودولو محدود بـ 32-بت: استعمل قيمة ≤ 2147483647.

### الكلمات المحجوزة
`صحيح`، `خطأ`، `لاشيء`، `تعداد` — لا تُستخدم كأسماء أصناف/متغيرات.
