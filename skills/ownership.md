# مهارة: الملكية والأعمار في لغة ص

نموذج على نمط Rust: **المفسّر** يقبل تعليقات العمر كمعلومات إضافية، و**المترجم** (`sad-build`) يفرض القيود.

## تمرير القيم
- القيم الأساسية والمصفوفات تُمرَّر **بالقيمة** إلى الدوال الحرة — التعديل داخل الدالة لا يؤثر على الأصل.
```sad
دالة بدّل(مصفوفة، ع1، ع2)   # ❌ لن يُعدّل الأصل
    ...
نهاية
```
- الحل: استعمل `&` للتمرير بالمرجع، أو غلّف الحالة في صنف.

## الاستعارة بالمرجع `&`
```sad
دالة زيادة(&عدد، مقدار)     # & في التعريف
    عدد = عدد + مقدار
نهاية

متغير س = 10
زيادة(&س، 5)                # & في الاستدعاء أيضًا
# س = 15
```

## تعليقات العمر (Lifetimes)
الفاصلة العليا `'` تلتصق باسم العمر مباشرة — **لا مسافة**.
```sad
دالة أطول<'أ>(س، ص)              # عمر واحد
    إذا (طول(س) > طول(ص)) ارجع س وإلا ارجع ص نهاية
نهاية

دالة دمج<'أ، 'ب>(أول، ثاني)      # عدة أعمار
    ارجع أول + " و " + ثاني
نهاية

بنية مرجع<'أ>                    # عمر على بنية
    رقم معرف = 0
نهاية
```

### أعمار خاصة
| العمر | المعنى |
|-------|--------|
| `'ثابت` | static — يعيش طوال البرنامج |
| `'_` | wildcard — يُترك للاستنتاج |

```sad
دالة ثابتة<'ثابت>()
    ارجع "أنا ثابتة"
نهاية

متغير س = 42
متغير م = &'أ س            # استعارة مع تعليق عمر
```

## الكتل غير الآمنة
```sad
@غير_آمن
    # عمليات تتجاوز فحوص الأمان
نهاية
```

## التأجيل المضمون (Defer)
`أجّل` يضمن تنفيذ تنظيف عند الخروج من الدالة، بترتيب LIFO، حتى عند الاستثناء، دون تغيير قيمة الإرجاع.
```sad
دالة مثال()
    أجّل اطبع_سطر("تنظيف!")
    اطبع_سطر("عمل...")
    # الخرج: عمل... ← تنظيف!
نهاية
```

## أخطاء شائعة
- `<أ>` بلا فاصلة عليا → خطأ؛ الصحيح `<'أ>`.
- `<' أ>` بمسافة → خطأ؛ الصحيح `<'أ>`.
- توقّع أن تُعدِّل دالة حرة مصفوفةً مُمرَّرة → لن يحدث؛ استعمل `&` أو صنفًا.
