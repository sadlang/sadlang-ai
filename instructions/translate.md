# تعليمات: الترجمة بين لغة ص ولغات أخرى

أنت وكيل يترجم الكود من/إلى لغة ص (غالبًا من Python/JS/Rust).

## جدول التقابل (الكلمات المفتاحية)
| لغة ص | إنجليزي |
|-------|---------|
| `دالة` | function/def |
| `ارجع` | return |
| `متغير` / `ثابت` | var/let / const |
| `صنف` / `بنية` | class / struct |
| `إذا` / `وإلا` / `وإلا_إذا` | if / else / elif |
| `بينما` | while |
| `لكل ... في` | for ... in |
| `اخرج` / `تابع` | break / continue |
| `نهاية` | } / end / dedent |
| `جديد` | new |
| `هذا` | this/self |
| `باني` | constructor/__init__ |
| `يرث` / `الأساس` | extends / super |
| `حاول/امسك/ارمي/أخيراً` | try/catch/throw/finally |
| `صدّر/استورد/من` | export/import/from |
| `صحيح/خطأ/لاشيء` | true/false/null |
| `طابق/عندما` | match/when/case |
| `لامدا` | lambda |
| `سمة/نفّذ` | trait/interface / implements |
| `عامل` | operator |
| `اطبع/اطبع_سطر` | print / println |

## قواعد الترجمة إلى لغة ص
1. حوّل كل كتلة (مسافة بادئة أو `{}`) إلى كتلة مُغلَقة بـ`نهاية`.
2. عناصر القوائم بالفاصلة العربية `،`.
3. f-strings → `م"... {متغير}"`.
4. إنشاء كائن → أضف `جديد`.
5. التمرير بالمرجع → `&` في التعريف والاستدعاء.
6. استعمل أسماء دوال المكتبة الدقيقة (`skills/stdlib.md`).
7. حافظ على المنطق والتعليقات؛ ترجم التعليقات للعربية.

## قواعد الترجمة من لغة ص
- اعكس الجدول؛ استنتج النوع من السياق إذا كانت اللغة الهدف مُصنَّفة.
- نبّه إن استُعملت ميزة لا مقابل مباشر لها (أعمار، عقود `يتطلب/يضمن`، ماكروز، `أجّل`).
