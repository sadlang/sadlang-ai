# مهارة: نظام الأنواع في لغة ص

أنواع باستنتاج مع إمكانية التصريح الصريح. أسماء الأنواع **ليست محجوزة** — يمكن استخدامها كمعرّفات
(لكن عندها لا تصلح كنوع في النطاق نفسه).

## الأنواع الأساسية
| النوع | المعنى | مثال |
|-------|--------|------|
| `رقم` | عدد صحيح | `رقم ع = 25` |
| `عشري` | عدد عشري | `عشري د = 98.5` |
| `نص` | سلسلة نصية | `نص اسم = "أحمد"` |
| `منطقي` | true/false | `منطقي نشط = صحيح` |

`نوع(قيمة)` يُرجع اسم النوع نصًّا: `نوع(42)` → `"رقم"`. التحويل: `لرقم(...)`، `لنص(...)`.

> القيم المركّبة (مصفوفة، خريطة، كائن) نوعها `"كائن"`. عناصر واجهة المستخدم في لغة ص كائنات عمدًا.

## الاستنتاج مقابل التصريح
```sad
متغير س = 5          # يُستنتج رقم
رقم ص = 5            # صريح
دالة رقم جمع(رقم أ، رقم ب)   # توقيع كامل بأنواع
    ارجع أ + ب
نهاية
```

## البرمجة الكائنية
```sad
صنف حساب
    نص صاحب
    رقم رصيد
    ساكن رقم العدد = 0       # عضو ساكن — الوصول عبر اسم الصنف فقط

    باني(نص صاحب، رقم رصيد)
        هذا.صاحب = صاحب
        هذا.رصيد = رصيد
        حساب.العدد = حساب.العدد + 1
    نهاية

    دالة ايداع(رقم مبلغ)
        هذا.رصيد = هذا.رصيد + مبلغ
    نهاية
نهاية
```

### الوراثة وتعدد الأشكال
```sad
صنف توفير يرث حساب
    باني(صاحب، رصيد، نسبة)
        الأساس(صاحب، رصيد)      # الأساس = super (أو أساس)
        هذا.نسبة = نسبة
    نهاية
    دالة ايداع(مبلغ)            # تجاوز كامل
        هذا.رصيد = هذا.رصيد + مبلغ + (مبلغ * هذا.نسبة)
    نهاية
نهاية
```
> ملاحظة: `الأساس(...)` موثَّق لاستدعاء **باني** الأب فقط. لا توجد صيغة موثَّقة لاستدعاء دالّة
> الأب المُتجاوَزة من داخل التجاوز (مثل `الأساس.ايداع(...)`) — أعِد التنفيذ كاملًا في التجاوز.

### محددات الوصول
`عام` (افتراضي) · `خاص` · `محمي` (للوارثين).

### تحميل العوامل
```sad
صنف متجه
    عامل +(آخر)
        ارجع جديد متجه(هذا.س + آخر.س، هذا.ص + آخر.ص)
    نهاية
نهاية
```

### السمات (Traits)
```sad
سمة قابل_للطباعة
    دالة اطبع()
نهاية
نفّذ قابل_للطباعة لـ كتاب
    دالة اطبع()
        اطبع_سطر("كتاب: " + هذا.العنوان)
    نهاية
نهاية
```

### الخصائص (Properties)
```sad
خاصية العمر
    احصل
        ارجع _العمر
    نهاية
    عيّن(قيمة)
        إذا (قيمة >= 0) _العمر = قيمة نهاية
    نهاية
نهاية
```

## قيود الأنواع (Where) — مُفرضة وقت التشغيل
```sad
دالة مضاعفة(أ)
    حيث ت: رقمي
    ارجع أ * 2
نهاية
مضاعفة("نص")   # RuntimeError: فشل قيد النوع
```
السمات المُفرضة فعليًا: `رقمي` (أعداد صحيحة/عشرية)، `نصي` (نصوص)، `قابل_للمقارنة` (أرقام أو نصوص).
سمات أخرى = إعلامية فقط (لا فحص).

## ملاحظات مهمة
- إنشاء الكائنات يتطلب `جديد`.
- البنية `بنية` لا تدعم باني ولا دوال — حقول فقط؛ استخدم دالة مصنّعة.
- العضو الساكن يُوصَل عبر `اسم_الصنف.الحقل` لا عبر `هذا`.
- الكلمات المحجوزة: `صحيح`، `خطأ`، `لاشيء`، `تعداد` — لا تُستخدم كأسماء.

انظر [ownership](ownership.md) للأعمار والاستعارة، و[errors](errors.md) للمزالق.
