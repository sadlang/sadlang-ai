# مهارة: قواعد لغة ص النحوية

معرفة مرجعية مكثّفة لتوليد كود لغة ص صحيح نحويًا. كل كتلة تُغلَق بـ`نهاية`. لا أقواس معقوصة `{}`.

## التعليقات
```sad
# تعليق سطري
#* متعدد الأسطر *#
## توثيقي
```

## المتغيرات والثوابت
```sad
متغير اسم = "أحمد"      # استنتاج النوع
رقم عمر = 25            # نوع صريح: رقم، عشري، نص، منطقي
ثابت باي = 3.14159      # تعديل الثابت ⇒ خطأ وقت تشغيل
```

## القيم الخاصة
`صحيح`/`صح` = true · `خطأ`/`خاطئ` = false · `لاشيء`/`عدم` = null

## العوامل
- حسابية: `+ - * / % **`
- مقارنة: `== != < > <= >=`
- منطقية: `&&`/`و` · `||`/`أو` · `!`/`ليس`
- إسناد: `= += -= *= /=`
- عضوية: `3 في [1، 2، 3]`
- ثلاثي: `شرط ? قيمة1 : قيمة2`

> فاصلة العناصر هي الفاصلة العربية `،`.

## التحكم بالتدفق
```sad
إذا (شرط)
    # كود
والا_اذا (شرط2)
    # كود
وإلا
    # كود
نهاية

بينما (شرط)
    # كود
نهاية

لكل عنصر في [1، 2، 3]
    اطبع_سطر(عنصر)
نهاية

لكل ع في 1..10          # نطاق
    اطبع_سطر(ع)
نهاية
```
- بدائل: `اذا`=`إذا` · `والا`=`وإلا` · `والا_اذا`=`وإلا_إذا`
- حلقات: `اخرج`/`توقف` = break · `تابع`/`استمر` = continue

### مطابقة الأنماط
```sad
طابق (قيمة)
    عندما 1:        اطبع_سطر("واحد")
    عندما 1..10:    اطبع_سطر("نطاق")
    عندما أ || ب:   اطبع_سطر("أو")
    عندما ن @ 1..10: اطبع_سطر("ربط")
    عندما _:        اطبع_سطر("آخر")
نهاية
```
أنماط: حرفي، شامل `_`/`افتراضي`، متغير، قائمة `[أ، ب]`، OR، نطاق، ربط `@`.

## الدوال
```sad
دالة جمع(أ، ب)              # استنتاج الإرجاع
    ارجع أ + ب
نهاية

دالة رقم جمع(رقم أ، رقم ب)   # أنواع صريحة
    ارجع أ + ب
نهاية

دالة رئيسية()                # نقطة دخول اختيارية
نهاية
```
- لامدا: `لامدا(س) => س * 2` (السهم فقط، لا كتلة)
- إغلاقات تلتقط النطاق المحيط.
- تمرير بالمرجع: `&` في التعريف وفي الاستدعاء — `دالة زيادة(&عدد، مقدار)` ثم `زيادة(&س، 5)`

## النصوص والمجموعات
```sad
اطبع_سطر(م"مرحباً {اسم}!")     # نص منسّق (f-string) ببادئة م

متغير م = [1، 2، 3]            # مصفوفة
متغير خ = {"اسم": "أحمد"}      # خريطة
```
تفصيل الدوال في [stdlib](stdlib.md).

## البنى والأصناف
```sad
بنية نقطة          # حقول فقط — لا باني ولا دوال
    رقم س = 0
    رقم ص = 0
نهاية

صنف حيوان           # كائني كامل
    باني(الاسم)
        هذا.اسم = الاسم
    نهاية
    دالة صوت()
        ارجع "..."
    نهاية
نهاية
متغير ق = جديد حيوان("قط")     # جديد إلزامي
```
تفصيل الوراثة والسمات في [types](types.md).

## الوحدات
```sad
صدّر دالة جمع(أ، ب)            # صدّر للتصدير
    ارجع أ + ب
نهاية
من رياضيات استورد جمع كـ أضف   # استيراد بأسماء بديلة
استورد رياضيات                 # وحدة كاملة
```

## ميزات متقدمة
- **ماكروز**: تعريف `ماكرو`، استدعاء بـ`!` إلزاميًا، يجب التعريف قبل الاستدعاء.
- **طرق الامتداد**: `امتداد صنف ... نهاية` — لا يجوز تجاوز دالة موجودة.
- **أعمار**: `<'أ>` فاصلة عليا ملتصقة بالاسم؛ خاصة: `'ثابت`، `'_`. تفصيل في [ownership](ownership.md).
- **عقود**: `يتطلب`/`يضمن` (المتغير `النتيجة` متاح في `يضمن`).
- **عقود ذكية**: `عقد` كالصنف لكن لا يرث إلا من عقد.
- **اختبار خصائص**: `اختبر("...") تكرارات 100 بذرة 42` مع `_تكرار`، `_عشوائي`.
- **تأجيل**: `أجّل` تنفيذ LIFO عند الخروج.
- **توجيهات**: `@حجم`، `@ذري`، `@متطاير`، `@غير_آمن`، `@وقت_الترجمة`.

تفصيل المزالق في [errors](errors.md).
