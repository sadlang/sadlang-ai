# مهارة: المكتبة القياسية ودوال لغة ص المدمجة

## الإدخال/الإخراج
| الدالة | الوصف |
|--------|-------|
| `اطبع(قيمة)` | طباعة بدون سطر جديد |
| `اطبع_سطر(قيمة)` | طباعة مع سطر جديد |
| `اقرأ()` | قراءة نص |
| `اقرأ_رقم()` | قراءة رقم |

## الرياضيات
`جذر(ن)`/`جذر_تربيعي(ن)` · `أس(قاعدة، أس)` · `مربع(ن)` · `مطلق(ن)` ·
`تقريب(ن)` · `أرضية(ن)` · `سقف(ن)` · `جيب/جيب_تمام/ظل(ن)` · `باي()` · `عشوائي()` (0–1).

## النصوص
| دالة/طريقة | الوصف |
|-----------|-------|
| `طول(نص)` | الطول |
| `تحويل_كبير/تحويل_صغير(نص)` | حالة الأحرف |
| `استبدل(نص، قديم، جديد)` | استبدال |
| `استخراج(نص، بداية، نهاية)` | substring |
| `قص_أطراف(نص)` | trim |
| `يبدأ_بـ` / `ينتهي_بـ` | بادئة/لاحقة (تاء مربوطة) |
| `يحتوي(نص، فرعي)` | contains (لا `يحتوي_على`) |
| `بحث(نص، فرعي)` | indexOf |

طرق النقطة متاحة: `"Hello".تحويل_صغير()`، `ن.بحث("ell")`.

## المصفوفات
| دالة / طريقة | الوصف |
|--------------|-------|
| `طول(م)` / `م.الطول()` | العدد |
| `أضف(م، عنصر)` / `م.اضف(...)` | إضافة للنهاية |
| `أزل(م، فهرس)` / `م.احذف(فهرس)` | إزالة بالفهرس |
| `أول(م)` / `آخر(م)` | أول/آخر |
| `فرز(م)` / `م.رتب()` | ترتيب |
| `قلب(م)` / `م.اقلب()` | عكس (لا `اعكس`) |
| `شريحة(م، بداية، نهاية)` | جزء |
| `م.احذف_اخير()` / `م.انزع()` | pop (لا `احذف_الاخير`) |
| `م.ادخل(فهرس، عنصر)` | إدخال |
| `م.صل("، ")` | join (لا `دمج`) |

### الدوال العليا
```sad
تخطيط([1، 2]، لامدا(ن) => ن * 2)      # map → [2، 4]
تصفية([1، 2، 3]، لامدا(ن) => ن > 1)   # filter → [2، 3]
اختزال([1، 2، 3]، "جمع"، 0)           # reduce → 6
```

## الخرائط
| الطريقة | الوصف |
|---------|-------|
| `خ.الحجم()` | العدد |
| `خ.مفاتيح()` / `خ.القيم()` | المفاتيح/القيم |
| `خ.يحتوي("مفتاح")` | فحص وجود |
| `خ.فارغة()` / `خ.امسح()` | فارغة / حذف الكل |
| `خ.نسخ()` / `خ.دمج(أخرى)` | نسخ / دمج |
| `خ.احذف("مفتاح")` / `خ.عناصر()` | حذف / أزواج |

## التحويل والفحص
`لرقم(قيمة)` · `لنص(قيمة)` · `نوع(قيمة)` (يُرجع اسم النوع نصًّا).

## الملفات
`اقرأ_ملف(مسار)` · `اكتب_ملف(مسار، محتوى)` · `احذف_ملف(مسار)`.

> راجع [errors](errors.md) للأسماء الدقيقة التي يخطئ فيها الوكلاء كثيرًا.
