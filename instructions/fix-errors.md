# تعليمات: إصلاح أخطاء كود لغة ص

أنت وكيل يشخّص ويصلح كود لغة ص.

## خطوات التشخيص
1. **افحص الأقواس والإغلاق**: هل كل كتلة مغلقة بـ`نهاية`؟ هل توجد `{}` خاطئة؟
2. **افحص الكلمات**: `return`→`ارجع`، `break`→`اخرج`، أقواس `امسك (خ)`→`امسك خ`.
3. **افحص الإنشاء**: هل نُسي `جديد`؟ هل `أصل`/`أساس` بدل `الأساس`؟
4. **افحص أسماء المكتبة**: `احذف_الاخير`→`احذف_اخير`، `اعكس`→`اقلب`، `يحتوي_على`→`يحتوي`، `دمج`(للمصفوفة)→`صل`.
5. **افحص الدلالة**: تعديل مصفوفة مُمرَّرة لدالة حرة (تُمرَّر بالقيمة)؛ عضو ساكن عبر `هذا`.
6. **افحص الميزات**: ماكرو بلا `!` أو مُستعمل قبل تعريفه؛ لامدا ككتلة؛ `...` ليس الأخير.

## جدول الأخطاء ↔ الإصلاح
| العَرَض | السبب | الإصلاح |
|---------|-------|---------|
| "نسيت نهاية" / كتلة مفتوحة | كتلة غير مغلقة | أضف `نهاية` |
| الدالة غير موجودة عند استدعاء ماكرو | نسيان `!` | استعمل `اسم!(...)` |
| RuntimeError: فشل قيد النوع | `حيث ت: رقمي` ومُرِّر نص | مرّر النوع الصحيح |
| modulo by zero عند رقم كبير | تجاوز 32-بت | قيمة ≤ 2147483647 |
| String index must be integer | تقطيع نص بنطاق | استعمل حلقة |
| الطريقة غير موجودة | اسم مكتبة خاطئ | راجع `skills/stdlib.md` |
| RuntimeError وراثة العقد | عقد يرث صنفًا عاديًا | اجعله يرث عقدًا |
