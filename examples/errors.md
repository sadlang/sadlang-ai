# أمثلة: معالجة الأخطاء والمزالق

## حاول/امسك/أخيراً
```sad
دالة اقسم(أ، ب)
    حاول
        إذا (ب == 0)
            ارمي "القسمة على صفر"
        نهاية
        ارجع أ / ب
    امسك خ                    # بدون أقواس
        اطبع_سطر("خطأ: " + خ)
        ارجع 0
    أخيراً
        اطبع_سطر("انتهت العملية")
    نهاية
نهاية
اطبع_سطر(اقسم(10، 0))
```

## التقاط حسب النوع
```sad
حاول
    ارمي جديد خطأ_شبكة("انقطاع الاتصال")
امسك خطأ_شبكة خ
    اطبع_سطر("شبكة: " + خ)        # استعمل خ مباشرة — لا خاصيّة .رسالة موثَّقة
امسك خ
    اطبع_سطر("عام: " + خ)
نهاية
```

## العقود
```sad
دالة قسمة_آمنة(أ، ب)
    يتطلب ب != 0
    يضمن النتيجة >= 0
    ارجع مطلق(أ / ب)
نهاية
اطبع_سطر(قسمة_آمنة(10، 2))   # 5
# قسمة_آمنة(10، 0)           # RuntimeError: انتهك الشرط المسبق ب != 0
```

## مزالق وإصلاحها

### قبل ❌ / بعد ✅ — نسيان جديد
```sad
متغير ش = حيوان("قط")        # ❌
متغير ش = جديد حيوان("قط")   # ✅
```

### باني الأب
```sad
صنف قطة يرث حيوان
    باني(اسم)
        أصل(اسم)             # ❌ أصل غير معروفة
        الأساس(اسم)          # ✅
    نهاية
نهاية
```

### امسك بأقواس (قديم)
```sad
امسك (خ)                     # ❌ لم يعد مدعومًا
امسك خ                       # ✅
```

### أسماء دوال المكتبة الدقيقة
```sad
م.احذف_الاخير()              # ❌  →  م.احذف_اخير()  ✅
م.اعكس()                     # ❌  →  م.اقلب()        ✅
"نص".يحتوي_على("ص")          # ❌  →  "نص".يحتوي("ص") ✅
مصفوفة.دمج("، ")             # ❌  →  مصفوفة.صل("، ") ✅
```

### تقطيع النص غير مدعوم
```sad
متغير جزء = نص[0..5]         # ❌ String index must be integer
# ✅ استعمل حلقة
متغير جزء = ""
لكل ع في 0..5
    جزء = جزء + نص[ع]
نهاية
```

### اختبار خصائص
```sad
اختبر("الجمع تبادلي") تكرارات 100 بذرة 42
    متغير أ = _تكرار + 1
    متغير ب = _تكرار * 2
    إذا (أ + ب != ب + أ)
        ارمي "ليس تبادليًا"
    نهاية
نهاية
```
